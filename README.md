# QuoteFlow 🌟

A sleek, modern, and interactive Random Quote Generator app built with Flutter. QuoteFlow goes beyond a simple local list by fetching real-time inspirational quotes directly from a live web API, letting users save beautiful quote cards directly to their photo gallery, and allowing instant quote-sharing with friends.

Developed as part of the **CodeAlpha App Development Internship (Task 2: Random Quote Generator)**.

## ✨ Features

*   **Dynamic API Integration:** Fetches fresh, high-quality quotes and authors in real-time using asynchronous HTTP requests.
*   **Minimal & Elegant UI:** Clean user interface with rich typography (using Playfair Display and Plus Jakarta Sans fonts) and fluid animations.
*   **Save Quote as Image:** Uses advanced pixel rendering to capture the beautiful quote card and save it directly to the user's device gallery.
*   **Seamless Sharing:** Integrated social sharing to instantly send text quotes to messaging apps or social media platforms.
*   **Robust Error Handling:** Smooth fallback system to ensure the app continues to perform gracefully offline.

---

## 🛠️ Tech Stack & Dependencies

This project is built using **Flutter** and **Dart**, leveraging the following packages:

*   **`http`** - For fetching real-time quotes from online APIs.
*   **`share_plus`** - For triggering native system share sheets to share quote text.
*   **`gal`** - For seamless, cross-platform saving of rendered quote cards to the device gallery.
*   **`google_fonts`** - For loading custom premium typography.
